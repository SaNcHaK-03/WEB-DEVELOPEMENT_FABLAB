Web Developement

HTML, CSS & JavaScript

01DAY

Basic HTML Tags

Started with core HTML tags to structure a webpage. Built a page listing SSGMCE branches and semester subjects using text and list tags.

  • <p>, <b>, <i>, <marquee> — text formatting
  • <ul>, <ol>, <li> — lists
  • bgcolor, font size, <br> attributes
02DAY

Image, Anchor, Video/Audio & Table

Embedded media into a webpage and structured data using tables. Added a YouTube link, local image, audio player, and autoplay video.

  • <img>, <a href> — image and hyperlink
  • <audio controls>, <video autoplay loop muted>
  • <table>, <tr>, <td>, <th> — data tables
03DAY

HTML Forms

Built a Committee Interview Registration Form with multiple input types and validation. Learned how forms collect and submit user data.

  • <form> with action, method, enctype
  • Input types: text, password, email, tel, date, url, color, range, radio
  • Attributes: placeholder, required, minlength, pattern
04DAY

Header, Nav, Footer with CSS Selectors

Built an "Engineering World" club page using semantic HTML. Styled it using various CSS selectors and hover effects on navigation links.

  • <header>, <nav>, <section>, <footer> — semantic tags
  • CSS selectors: element, class, id, universal, group
  • :hover pseudo-class for nav links
05DAY

Box Model & Flexbox

Learned CSS layout using the Box Model and Flexbox. Centered a To-Do card on screen and understood how spacing and sizing work around elements.

  • Box Model: margin, border, padding, content
  • display: flex, justify-content, align-items
  • height: 100vh, border-radius, background-color
06DAY

JavaScript Basics & Operations

Introduction to JavaScript variables and operators. Practiced all arithmetic operations and explored string concatenation and boolean behaviour in the console.

  • var, console.log() — variables and output
  • Arithmetic: +, -, *, /, % operators
  • String concatenation, boolean ops, NaN
07DAY

To-Do List Project

Final project combining HTML, CSS, and JavaScript. Built a functional To-Do List where users can add and delete tasks dynamically using DOM manipulation.

  • DOM manipulation — add/delete tasks dynamically
  • Flexbox UI layout and card styling
  • Linking external script.js to HTML
